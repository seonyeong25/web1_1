HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup lan guage for creating web pages and web application.

The html, which is known as the root element, wraps the contents of a web page and it can be found immediately after the doctype declaration.

Metadata contains information about the page like styles, scripts and data to help search engines or browsers to use and render the page.
